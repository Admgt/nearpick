# NearPick GUI/UX dokumentáció

Ez a mappa a NearPick alkalmazás GUI/UX beadási csomagját tartalmazza. A dokumentáció a Flutter kliens aktuális képernyőire, a Firebase/Cloud Functions adatforrásokra és a `docs/assets/ux/` alatt rögzített screenshot evidence-re épül.

- [Önértékelés](self_assessment.md)
- Screenshotok: [screenshots/](screenshots/)
