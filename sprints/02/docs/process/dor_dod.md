# DoR / DoD – Sprint 2 

## Definition of Ready (DoR)
- Story INVEST szerint megfogalmazva; AC-ek készek (Given–When–Then vagy rövid AC formátum)
- Üres állapot és hibaállapot definiálva minden releváns flow-ra
- Kapcsolódó UI vázlat / wireframe elérhető
- Elfogadási teszt (AC) automatizálási terv megvan

## Definition of Done (DoD)
- Unit és acceptance tesztek zöldek; Coverage ≥ 60% (coverage.xml)
- Smoke teszt zöld (smoke.yaml + smoke.http, preview környezet)
- ADR-ek frissítve (0001–0003), a Sprint 2-höz szükséges döntésekkel
- Traceability tábla aktualizálva (traceability.md)
- Terraform: validate + plan sikeres; plan.out artefakt elmentve
- PR: részletes leírás, screenshot vagy GIF, teszt- és build-riport linkek
